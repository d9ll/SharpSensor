# 🛰️ SharpSensor Library

![Arduino IDE](https://img.shields.io/badge/Arduino-IDE-00979D?style=for-the-badge&logo=arduino&logoColor=white)

Библиотека для удобной работы с инфракрасными датчиками расстояния **Sharp**. Она автоматизирует преобразование напряжения в расстояние и предоставляет инструменты для сглаживания данных.

## 🚀 Основные возможности
* **Поддержка моделей:** работа с основными датчиками линейки GP2Y.
* **Усреднение значений:** встроенная функция для фильтрации шумов.
* **Калибровка:** возможность настройки под конкретный экземпляр датчика.
* **Гибкость:** вывод данных в см, мм, дюймах или сырых значениях АЦП.

## 📐 Поддерживаемые модели
| Модель | Диапазон |
| :--- | :--- |
| **GP2Y0A02YK0F** | 20 – 150 см |
| **GP2Y0A21YK0F** | 10 – 80 см |
| **GP2Y0A41SK0F** | 4 – 30 см |
| **GP2Y0A710K0F** | 100 – 550 см |

## 📦 Установка
1. Скачайте библиотеку в формате `.zip`.
2. В Arduino IDE выберите: `Скетч` -> `Подключить библиотеку` -> `Добавить .ZIP библиотеку...`.
3. Выберите скачанный файл.

## 💻 Пример использования
```cpp
#include <SharpSensor.h>

SharpSensor sensor; 

void setup() {
    Serial.begin(9600);
    
    // Настройка модели и пина
    sensor.model(GP2Y0A21YK0F); 
    sensor.attach(A0);
    
    // Установка количества образцов для усреднения
    sensor.setAverage(10);
}

void loop() {
    float distance = sensor.getCM();
    Serial.print("Расстояние: ");
    Serial.print(distance);
    Serial.println(" см");
    delay(100);
}
