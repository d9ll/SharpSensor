# 🛰️ SharpSensor Library

![Arduino IDE](https://img.shields.io/badge/Arduino-IDE-00979D?style=for-the-badge&logo=arduino&logoColor=white)
![License](https://img.shields.io/badge/license-MIT-blue?style=for-the-badge)

Удобная и точная библиотека для работы с инфракрасными датчиками расстояния **Sharp**. Она берет на себя всю математику преобразования напряжения в сантиметры и позволяет откалибровать датчик одной командой.

## 🚀 Основные возможности
* **Поддержка 4 популярных моделей:** от 4 см до 5.5 метров.
* **Усреднение значений:** сглаживание показаний для устранения шумов.
* **Калибровка:** легкая настройка под конкретный экземпляр датчика.
* **Гибкость:** получение данных в см, мм, дюймах или сырых значениях АЦП.

## 📐 Поддерживаемые датчики
| Модель | Диапазон |
| :--- | :--- |
| **GP2Y0A02YK0F** | 20 – 150 см |
| **GP2Y0A21YK0F** | 10 – 80 см |
| **GP2Y0A41SK0F** | 4 – 30 см |
| **GP2Y0A710K0F** | 100 – 550 см |

## 📦 Установка
1. Скачайте репозиторий как `.zip` архив.
2. В Arduino IDE выберите: `Скетч` -> `Подключить библиотеку` -> `Добавить .ZIP библиотеку...`.
3. Выберите скачанный файл.

## 💻 Пример использования
```cpp
#include <SharpSensor.h>

SharpSensor sensor; 

void setup() {
    Serial.begin(9600);
    
    // Настройка модели и пина
    sensor.model(GP2Y0A21YK0F); 
    sensor.attach(A0);
    
    // Опционально: включить усреднение по 20 замерам
    sensor.setAverage(20);
}

void loop() {
    float distance = sensor.getCM();
    Serial.print("Расстояние: ");
    Serial.print(distance);
    Serial.println(" см");
    delay(100);
}
